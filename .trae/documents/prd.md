## 1. 产品概述

统一服务管理平台，将日志监控系统（Log Monitor）和服务拨测大盘（Dashboard）合二为一，提供统一认证、统一存储、统一前端。
- 目标用户：运维工程师、后端开发
- 解决核心痛点：无需在多个服务间切换，一个端口一个页面管理所有服务和日志

## 2. 核心功能

### 2.1 功能模块
1. **服务大盘 Dashboard**：主机管理 + 拨测状态 + SSH 终端 + 角色管理 + Excel 导出
2. **日志监控 Log Monitor**：ES 日志查询 + 告警规则管理 + 飞书 Webhook 配置 + 实时日志流(WS) + 限流日志管理
3. **统一认证**：JWT 单点登录、用户管理
4. **持久化存储**：SQLite/MySQL 双驱动

### 2.2 页面详情
| 页面 | 模块 | 功能说明 |
|------|------|---------|
| 服务大盘 | 主机列表 | 主机状态 LED 灯展示(青/绿/红/黄/蓝)，资源信息，维护模式 |
| 服务大盘 | 角色导航 | 按角色类型筛选，角色模版管理 |
| 服务大盘 | SSH 终端 | xterm.js WebSocket 终端，凭据管理 |
| 服务大盘 | 统计概览 | 正常/异常/未知主机数量统计 |
| 日志监控 | 实时日志流 | WebSocket 实时推送匹配/未匹配日志 |
| 日志监控 | 告警规则 | CRUD 告警规则，冷却时间配置 |
| 日志监控 | 飞书配置 | Webhook URL、@提及、限流参数配置 |
| 日志监控 | 统计面板 | 限流状态、去重缓存、告警次数统计 |

## 4. 界面设计

### 4.1 设计风格
- **主色调**：深色主题为主（#0f172a 深蓝黑），强调色使用亮绿(成功)和橙红(告警)
- **布局**：顶部固定导航栏 + 标签页切换 + 卡片式内容区
- **字体**：系统无衬线字体，代码块等宽字体
- **风格**：现代、简洁、信息密度适中，LED 指示灯表示状态

### 4.2 页面元素
| 页面 | 模块 | UI 元素 |
|------|------|---------|
| 服务大盘 | 主机卡片列表 | 状态指示灯(圆点)+IP+主机名+资源+操作按钮 |
| 服务大盘 | 角色筛选导航 | 横向按钮组，选中高亮 |
| 服务大盘 | SSH 终端 | xterm.js 黑底绿字 + 输入框 |
| 日志监控 | 日志流 | 左边框着色(红/橙/灰)行式列表 |
| 日志监控 | 统计卡片 | 四格栅格 + 大数字 + 标签 |

### 4.3 响应式
桌面端优先，最小支持 1024px 宽度。
